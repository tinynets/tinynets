In a world of LLMS and SOTAs with hundreds of billions of parameters, the compute and infrasturcture needed to build and train massive models becomes increasingly unreachable.

With the tinynets project we hope to accomplish 2 objectives:

1. Be a place to store simple data science projects that can serve as a reference for new comers to the field.
2. Write code and utilities in the pursuit of having a friendlier data science workflow.

Within the tinynets project, over the coming weeks and months you can expect to find small, concise projects that demonstrate various ML principles (not neccasarily always including an ANN).

For more information please visit http://tinynets.io
